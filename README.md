
# ORGTerm
Controls:
```
\	volume up
.	volume down
space	pause/resume
q	quit
```
how to build:

```
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build
```

get Cave Story orgs using the included installorgs.sh file - requires curl and unzip
```
./installorgs.sh
```
outputs to ./orgs directory
